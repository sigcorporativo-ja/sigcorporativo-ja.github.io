Notas versión 6.1.0

API js

  • Actualización OpenLayer v6.9.0.
  • Nueva clase M.Style.Generic, usable con cualquier capa vectorial, incluyendo las que contengan elementos de diferentes tipos de geometrías.
  • Evento M.evt.CHANGE_ZOOM solo se disparará cuando haya cambio efectivo de nivel de zoom.
  • Mejora del comportamiento de M.exception.
  • Renombrado el método de acceso al nivel de OL.
  • Nuevas opciones de parametrización del comportamiento del etiquetado vectorial del plugin printerdos.
  • Corregido bug getCentroid de Feature.

El código fuente se encuentra disponible en GitHub, y la wiki se irá actualizando para recoger estas novedades en los próximos días.
