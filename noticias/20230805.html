Notas versión 6.4.0

  • Actualización a Open Layers v7.2.2
  • Nuevo evento de cambio de visibilidad de capa M.evt.LAYER_VISIBILITY_CHANGE
  • Añadido atributo "overflow" a la simbología label para permitir que el etiquetado sobrepase los polígonos
  • Nuevas funciones de mapa getOverlayLayers y removeOverlayLayers para obtener y eliminar capas no-base (overlay)
  • El estilo de Coropletas aplica por defecto borde negro para mejor visualización de elementos con mismo rango
  • Clase Vector implementa ahora el método getFeaturesExtent para correcto funcionamiento de algunos escenarios
  • Corregido atributo "hover" de simbología Cluster
